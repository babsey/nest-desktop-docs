================================

Get the source code
-------------------

The source code of NEST Desktop is hosted on `GitHub <https://github.com/nest-desktop/nest-desktop>`__.
Clone NEST Desktop from the GitHub repository:

.. code-block:: bash

  git clone https://github.com/nest-desktop/nest-desktop
  cd nest-desktop
